# 本地财务管理系统 — 使用说明

## 安装依赖

```bash
pip install PySide6 openpyxl
```

## 启动程序

```bash
python3 main.py
```

## 功能模块

### 📊 概览
- 自动统计本月收入、支出、利润
- 显示本月明细和客户汇总

### 👥 客户管理
- 添加/编辑/删除客户公司
- 存储税号、联系人、电话、地址
- 点击"查看账目"跳转到该客户记账页

### 📝 记账管理
- 手动录入每笔收支（日期、类型、金额、科目、摘要、凭证号）
- 按客户/年份/月份/类型筛选
- 底部实时显示收支汇总
- 支持导出 Excel

### 🔄 银行对账
- 导入银行流水 CSV 或 Excel
  - CSV 列名：日期、摘要、金额、余额
- 点击"自动匹配"按日期+金额与记账记录对账
- 左右对照查看银行流水和记账记录

### 📈 财务报表
- **利润表**：按科目分类汇总收支净额
- **资产负债表**：简版资产/负债/权益总览
- **月度收支表**：全年12个月环比分析
- 支持导出多 Sheet Excel 报表

## 数据文件

所有数据保存在同目录下的 `finance.db`（SQLite 数据库）

建议定期备份此文件！

## 打包成 exe（Windows）

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name 财务管家 main.py
```
